# Client Brief — fill once, hand to the AI

Copy this file into a new client project, fill it in, then give it to the AI
agent together with `BROCHURE-SITE-RULES.md`. The brief — not a fixed template
— decides which pages get built.

```text
BUSINESS
- Name:
- Type (villa / tavern / boat rental / studio / other):
- One-line tagline:
- Short description (2–3 sentences):

LOCATION
- Address / area:
- Service area:
- Map embed URL (if any):

CONTACT
- Phone:
- WhatsApp:
- Email:
- Social links:

PAGES NEEDED (list only what this business requires)
- home
- (examples: rooms, menu, gallery, fleet, trips, booking, contact, about)
- your custom slugs here:

CONTENT & MEDIA
- Real photos available? (yes/no + where)
- Need stock photos? (yes/no + style keywords)
- Languages (e.g. Greek + English):
- Special notes / claims to avoid / must-include trust signals:
```

## How to use

1. Duplicate this file as `CLIENT-BRIEF.md` in the new project.
2. Fill every field you can; leave unknown ones blank.
3. Tell the agent: "Build this brochure site from CLIENT-BRIEF.md and
   BROCHURE-SITE-RULES.md. Propose the site map first."
